## Stacks

This project used:

1. React (functional components)
2. Emotion

## Requirements

1. Node
2. yarn

## Instruction

To run in local, do:

1. Clone the repo
2. Install package, using `yarn install`
3. Run `yarn start`

You can also see live here: [https://shafiya.netlify.com/](https://shafiya.netlify.com/)
